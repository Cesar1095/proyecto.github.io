Desarrollo web en el 2017 + Tips

En esta conferencia se habló sobre las dos partes del desarrollo web y las herramientas que se utilizan en estas, por ejemplo, en la parte de Front-End podemos utilizar HTML + CSS + JS, y para Back-End DB, BL, WS (servidores).

También nos menciona los temas de progressive web app y single-page application.

Progressive web app está enfocado para que la pagina pueda estar en cualquier dispositivo.

Una Progressive Web App es:

  • Progresiva: funciona para todos los usuarios, sin importar la elección de navegador, porque está construida con mejora progresiva como principio central.
  • Adaptable: se adapta a cualquier factor de formulario, sea escritorio, móvil, tablet o lo que venga en el futuro.
  • Segura: emitida vía HTTPS para evitar intromisiones y para garantizar que el contenido no se haya manipulado.
  • Estilo app: al usuario le parece una app con interacciones y navegación estilo app, porque está construida con modelo de shell de app.

Single-page application (SPA) es una aplicación web o sitio web que está en una solo página para dar una experiencia fluida como una aplicación de escritorio.
