Machine learning

Machine learning es una rama de la inteligencia artificial.

Algunos ejemplos donde se utiliza la inteligencia artificial:

  • Netflix
  • Tesla
  • Amazon
  • Google

Otras ramas que podemos encontrar en la inteligencia artificial son:

  • Sistemas expertos
  • Visión por computación
  • Reconocimiento de diálogos

Machine learning se divide en:

  • Deep learning
  • Aprendizaje no supervisado
  • Aprendizaje supervisado

Machine lerning es un modelo matemático que optimiza una función.

Los algoritmos que maneja son:

  • K-means clustering
  • Association rules

Cuando hablamos de machine learning también tenemos que mencionar las redes neuronales, estas están biológicamente inspiradas y se utilizan para clasificación.

En resumen, las redes neuronales trabajan de la siguiente forma:

  • Introducimos datos
  • Calculamos probabilidades
  • Evaluamos
  • Repetimos hasta que el error sea mínimo y lleguemos a un resultado optimo

Deep learning es un termino comercial, solo se añade mas capas ocultas a una red neuronal. Dependerá de la dificultad de la tarea para usar mas de una capa.

Entre las aplicaciones están:

  • Colorear imágenes que son a blanco y negro
  • Añadir sonido a un video que originalmente no tiene sonido
  • Segmentación de objetos
  • Generación automática de texto
  • Hacer que la computadora aprenda a jugar de manera automática.
